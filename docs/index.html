Install

  1. Download the .vsix from the latest release.
  2. In VS Code: Extensions panel, then Install from VSIX.
  3. Pick the color theme with Ctrl+K Ctrl+T.
  4. Enable the icons: "workbench.iconTheme": "terminal-workbench-icons"
